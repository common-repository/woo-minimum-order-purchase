== Description ==
Adds a Easily Minimum Order Purchase Amount to Woocommerce

Just enter the amount and a error message.

== Installation ==

1. Upload `woocommerce-minim-order-purchase.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
